# **SuperOperatingSystem (SOS)**

**SuperOperatingSystem (SOS)** is a **lightweight, community-driven, XP-inspired OS** designed for **learning, development, and decentralized collaboration**. SOS introduces a **tokenized developer ecosystem** where contributions are rewarded, apps are validated, and users can participate in a secure, isolated network.

---

## **Key Features**

### **Core OS**

* Minimal footprint (<200 MB) for **potato laptops**
* **Openbox desktop** with Windows XP-style UI
* Lightweight **taskbar, start menu, and multi-window support**
* Modular kernel and OS core, supporting x86 and ARM CPUs

### **Developer Ecosystem**

* **SuperGeany IDE**: Custom Geany with **SuperCode (.su) support**
* Builder GUI for compiling `.su → .sex` applications
* **Sandboxed runtime** ensures isolated, secure execution
* Token-based reward system for validated contributions

### **Network & Community**

* **Decentralized node network** via RetroShare and GotYou app
* Public/private node identity and discovery
* Community-based app submission and voting mechanism
* **Consensus-driven ledger** for secure token transactions

### **Pre-installed Apps & Utilities**

* **Media Player:** MPV
* **Graphics:** MyPaint
* **File Manager:** PCManFM
* **Terminal:** LXTerminal
* **Archiving:** Xarchiver, tar/gzip/p7zip/unzip
* **Disk Management:** GParted
* **System Monitor:** LXTask, network CLI tools
* Developer tools: Geany (customized), Nano, Builder GUI

### **Ecosystem Wallet**

* **CoinBag Wallet** for tracking tokens
* Staking for app submission & voting
* Ledger replicated across network nodes for transparency

### **Learning & Gamified Contributions**

* Earn **SuperCoin / SuperToken** by coding `.su` apps
* Contributions validated by the community
* Sandbox and Builder GUI guide learning while earning

---

## **Goals**

* Provide a **general-purpose OS** suitable for learning and development
* Create a **futuristic OS ecosystem** that is **secure, lightweight, and community-driven**
* Encourage **developers to build, share, and earn** in a decentralized environment

---

## **Getting Started**

* Minimal install: <200 MB, runs on low-end hardware
* Clone the repo and follow **`docs/developer_guide.md`** for building SOS
* Use **scripts/build_all.sh** to compile kernel, system, apps, and desktop
* Join the **node network** with **scripts/setup_node.sh** for testing and contributions
